# ASPNETCoreTemplate  
ASP.NET Core Template with:  
- .NET 6  
- ASP.NET Core with Razor Pages pattern
- Docker  
- Configured NuGet for:
  - ASP.NET Core Identity for signup and login
  - Entity Framework Core  
- Configured libman for:  
  - Bootstrap 5
  - jQuery 3.6
  - jQuery Validate
  - jQuery Validation Unobtrusive
  - FontAwesome
